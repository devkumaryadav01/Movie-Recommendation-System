# 🎬 Movie Recommendation System

## 📌 Overview
This project recommends similar movies using Collaborative Filtering based on user ratings.

## 🚀 Features
- Movie Recommendation
- Correlation-based Similarity
- User Rating Analysis
- Interactive Movie Search

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Collaborative Filtering

## 📂 Dataset
- movies.csv
- ratings.csv

## ▶️ Run the Project
```bash
python app.py
```

Enter a movie name and the system will recommend similar movies.
